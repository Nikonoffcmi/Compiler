Постановка задачи

Структура — тип данных, определяемый разработчиком и служащий для представления каких-либо объектов. Структуры содержат набор полей, которые представляют различные атрибуты объекта. Каждое поле имеет название и тип данных, как переменная.

Для определения структуры применяются ключевые слова type и struct.

Формат записи:

type имя_структуры struct{

поля_структуры

}

Примеры:

Структура состоящая из двух полей — первое представляет тип string, второе представляет тип int:

type person struct{

name string

age int

}

Структура состоящая из трех полей — все они представляют тип int:

type Circle struct{

x float

y float

r float

}
